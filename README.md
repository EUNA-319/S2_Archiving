- Andriod Studio 4.1.1

- 사용 스택
  - Kotlin
  - FireBase

## 앱 소개
- 주요 화면
  * Home : 메인 페이지로 친구 목록과 최신 게시글이 보여진다.
  * Like : 좋아요 페이지로 좋아요를 누른 게시글만 보여진다.
  * User : 마이 페이지로 로그인, 회원가입 및 내 정보 수정과 관려된 페이지이다.
